Análise de gastos

📌 Objetivo

Neste projeto desenvolvi um sistema capaz de coletar, armazenar, analisar e visualizar dados financeiros em dashboards interativos.

🛠️ Tecnologias

Python (Pandas, Flask)

MySQL

Power BI


📊 Funcionalidades:

 Integração com banco de dados MySQL
 
 Análise automática de dados com Pandas
 
 Dashboard Web utilizando Flask
 
 Indicadores de desempenho (total, média e maior gasto)
 
 Dashboard profissional desenvolvido no Power BI
